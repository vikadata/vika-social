## Usage

### Java Version Requirement

Java 8+ is required to use sdk. not support Java 8 below

## Getting Started

### Installation

* Maven

```xml

<dependency>
    <groupId>cn.vika</groupId>
    <artifactId>vika-social</artifactId>
    <version>1.0.0</version>
</dependency>
```

* Gradle

```groovy
dependencies {
  ...
  implementation('cn.vika:vika-social:1.0.0')
}
```
